# Insurance-Portfolio-Performance-Claims-Analytics
📊 Insurance Claims & Premium Performance Dashboard
Power BI Project – Portfolio Profitability Analysis
📌 Project Overview

This dashboard analyzes insurance portfolio performance by comparing Total Premium, Total Claims, Active Policies, and Loss Ratio.

The objective is to evaluate product profitability and identify high-risk segments using structured KPI analysis.

📈 Key Metrics

Total Premium: 805K

Total Claims: 1M

Active Policies: 296

Loss Ratio: 173.74%

📊 Dashboard Components
1️⃣ Premium Analysis

Total Premium by Product

Total Premium by Partner

Health Insurance generates the highest premium contribution.

2️⃣ Claims Analysis

Total Claims by Product

Health Insurance also shows the highest claims amount.

🔍 Key Insight

The Loss Ratio of 173.74% indicates that claims significantly exceed premium income.
This suggests that the current portfolio structure is unprofitable and requires further investigation into pricing, underwriting, or risk exposure.
